# Student TaskManager

A professional task management application for students with AWS backend integration.

## Features

- User authentication with AWS Cognito
- Task management (create, read, update, delete)
- File attachments for tasks
- Calendar integration
- User profile settings
- Responsive design with animations

## Technologies Used

- React.js
- AWS Amplify
- AWS Cognito (Authentication)
- AWS DynamoDB (Database)
- AWS S3 (File Storage)
- AWS Lambda (Backend Functions)
- AWS API Gateway (REST APIs)
- Framer Motion (Animations)
- React Icons

## Setup Instructions

1. Clone this repository
2. Install dependencies: `npm install`
3. Initialize Amplify: `amplify init`
4. Add authentication: `amplify add auth`
5. Add API: `amplify add api`
6. Add storage: `amplify add storage`
7. Push to AWS: `amplify push`
8. Start development server: `npm start`

## Deployment

1. Build the app: `npm run build`
2. Publish to AWS: `amplify publish`

## Environment Variables

The `aws-exports.js` file is automatically generated by Amplify and contains all necessary configuration.
